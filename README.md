1. **Clone the Repositories**

```bash
git clone https://github.com/30526/travel-ease-client

```

````

2. **Install Dependencies**

```bash
# Client
cd travel-ease-client
npm install

# Server
cd travel-ease-server
npm install
```

3. **Environment Variables**
   Create a `.env` file in both client and server folders and add your Firebase and database credentials.

4. **Run Locally**

```bash
# Client
npm run dev

# Server
npm start
```
